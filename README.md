## ğŸ“– Tentang Proyek

**Prism** adalah aplikasi media sosial berbasis foto yang dibangun menggunakan Flutter. Proyek ini mendemonstrasikan implementasi arsitektur **Clean Architecture** (Feature-First) dengan manajemen state **BLoC**. 

Aplikasi ini mengonsumsi REST API lengkap untuk fitur autentikasi, postingan, story, dan interaksi sosial.
